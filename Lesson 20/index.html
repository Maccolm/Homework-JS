• Task 0

    Component diagram. Values from 0 to 100. The area is divided into 3 parts (the value in the first part is red fill, the value in the second part is blue, the value in the third part is green.)

  • Task 1

    Dynamic search. There is a list of employees and a search field. When entered, all that contain the specified fragment are displayed

  • Task 2

    A list of cars is given (make, year of manufacture, price). Form elements for filtering using a drop-down list (form the content of these drop-down lists depending on the given list).

  • Task 3

    Game "Guess the number". Two users enter numbers between 1 and 10. The computer generates a random number. The winner is the one whose number is closer to the one generated.

  • Task 4

    Develop Loader (title, value is specified). Simulate loading with a timer.

  • Task 5

    Develop a form for entering a login and password.

  • Task 6

    A wandering object. Some object moves randomly within the container. When it meets the edge of the container, it bounces off.

  • Task 7

    Develop an element for entering the quantity of the selected product according to the sample (minimum quantity 1, maximum quantity 10, if exceeded, block the button)

  • Task 8

    Translator. Mixed cards with words in English and Ukrainian are displayed to the user. The user gradually clicks on the cards (highlighted with a blue frame). If correct pairs of cards corresponding to one word are found, then we remove these cards. Otherwise, highlight it with a red frame and after a second, remove the frame.
